# Linux Security Labs (RHCSA)

This repository contains hands-on Linux security engineering labs focused on securing Red Hat Enterprise Linux (RHEL) systems.

## Focus Areas
- Linux system hardening and secure configuration
- SELinux enforcement and access control
- Log analysis and threat detection
- Incident response fundamentals
- Host-based security automation

## Technologies
RHEL / Rocky Linux  
SELinux, FirewallD, OpenSSH  
Bash scripting  
AIDE (File Integrity Monitoring)

## Purpose
These labs simulate real-world Linux security scenarios aligned with entry-level Security Engineer and Infrastructure Security roles.
